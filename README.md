# Etec - Exercícios de Web

Exercícios e projetos práticos desenvolvidos por Alex Pereira no 1º semestre do Técnico em Desenvolvimento de Sistemas (Etec Joaquim Ferreira do Amaral).

Foco em HTML5, CSS3 e JavaScript: estrutura semântica, estilização, formulários, tabelas e lógica no navegador.

---

## Projetos

### Fenix Baixaki
`HTML5` `CSS3`

Loja de jogos e serviços digitais com layout completo. Menu de navegação com dropdowns, sidebar de categorias com submenus, grid de produtos com links externos e footer. O projeto mais estruturado do semestre — feito em dupla com Yuri.

### Formulário de Cadastro
`HTML5` `CSS3`

Formulário com campos de texto, e-mail, telefone, radio buttons e textarea, estruturado com tabela. Validação nativa via atributo `required`.

### Formulário Básico
`HTML5` `CSS3`

Cadastro simples com campos de e-mail, nickname e senha. Background com imagem via CSS e estilização de botões com hover.

### Folha de Pagamento
`HTML5` `CSS3`

Tabela semântica com `thead`, `tbody`, `caption` e linha de total com `colspan`. Estilização com hover nas células.

### Loops JavaScript
`HTML5` `CSS3` `JavaScript`

Quatro funções acionadas por botões: tabuada completa de 1 a 10 com for aninhado, contador com while, do-while com break, e analisador de string que conta caracteres, vogais, números e espaços via regex.

### Gerador de Tabuada
`HTML5` `CSS3` `JavaScript`

Gera a tabuada de qualquer número via prompt, com loop de repetição controlado pelo usuário.
